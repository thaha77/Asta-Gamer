# 😈 Black Clover: Asta Web Game

Welcome to the **Black Clover: Asta Game**, a fast-paced 2D web-based animation game inspired by the legendary anime *Black Clover*. Defeat the Grim Reaper devil and prove them wrong!



---

## 🚀 Game Features
* **Devil Theme:** Dark, neon-red devilish UI designed for true Black Clover fans.
* **Asta (Devil Union):** Control Asta in his powerful anti-magic form.
* **Grim Reaper Boss:** Dodge the dangerous devil coming from the right side.
* **Live Score:** Watch your score increase every time you successfully jump over the enemy.

---

## 🛠️ Built With (Technologies Used)
This project is built from scratch using clean code to learn web development and English keywords:
1. **HTML5:** Created the base layout, game box, and characters.
2. **CSS3:** Styled the dark neon look, position management, and running animations (`@keyframes`).
3. **JavaScript:** Added game logic, movement controls, keyboard spacebar detection, and hit detection (collision).

---

## 🎮 How to Play
1. Download all files (`index.html`, `style.css`, `script.js`, and images) into one folder.
2. Double-click or open **`index.html`** using Google Chrome.
3. Press **SPACEBAR** or **CLICK** anywhere on the screen to make Asta Jump.
4. Dodge the Grim Reaper and build the highest score!

---

## 📖 English Words Learned from this Code
By building this game, we learned these essential coding words:
* `Width` & `Height`: Sets the physical size of objects.
* `Absolute Position`: Places an element exactly where we want it.
* `Animation`: Makes elements move smoothly on the screen.
* `Event Listener`: Waits for a keyboard click or mouse touch.
* `Collision`: Detects when two characters hit each other (Game Over).
