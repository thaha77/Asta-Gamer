#Asta Game
This is a web-based game project built for Hack club Macondo.
The goal of the game is to jump over devil,survive and get a high score.

How to Play
-Press the "Spacebar" or "Click"on the screen to make the character jump.
-Avoid the devil to keep the game going.
-If you hit an devil it is Game Over.

Technologies Used
-HTNL5 Canvas for game graphics
-CSS for basic styling
-JavaScript for game logic and physics
